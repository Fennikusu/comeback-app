# Comeback App 🎮💰

Application Flutter de paris entre amis sur l'esport. 

## 📌 Technologies utilisées :
- Flutter (mobile & web admin)
- API PHP + MySQL
- WAMP (WebSocket)

## 📂 Structure du projet :
- `mobile/` : Application mobile (Flutter)
- `admin-web/` : Interface admin (Flutter Web)
- `api-php/` : API PHP avec MySQL
